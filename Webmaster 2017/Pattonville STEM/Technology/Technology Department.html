Computer Science Principles

Jeremiah Simmons

Primarily uses Python along with various platforms and computer languages. Goals include the development of computational thinking, enthusiasm for computer science, and generally creativity and collaboration. Lessons include topics such as app development, visualization of data, cybersecurity, and simulation. Sub-units of teaching include algorithms and agile development, images and object oriented libraries, security along with cryptography, and Excel spreadsheet programming and cell manipulation. Projects include a Scratch game, an AppInventor Android App, a Python GUI, and a website.

AP Computer Science A

Jeremiah Simmons

Mainly uses the Java programming language and is part of the Advance Placement program made by the College Board. Gives emphasis to programming methodology and algorithm development. Specific topics include the different data classes and their hierarchies, arrays along with the List interface, objects, algorithms, and syntax, as well as arithmetic, logic, and control statements. Projects include a chatbot for learning in string manipulation and image manipulation to work with one and two-dimensional arrays.

Computer Science A

Jeremiah Simmons

Further understanding of computer development is done through Android App development for mobile platforms. Utilizes industry-standard tools such as Android Studio, Java, XML, and device emulators. Units of teaching include vanilla Android development, advanced Android features, the LibGDX game development framework, and independent projects that involve the implementation of geolocation, communication with a database, and the use of the camera, speakers, and microphone.

Computer Science Capstone

Jeremiah Simmons

As the culmination of knowledge gained from the previous three year-long courses, students work on a year-long project that is unique to each of them. Students start out by working with a client to determine an initial problem, research the problem, prototype a solution, and then propose the solution to the client. After revision, students use the rest of the school year to work on the project. Focus is given to collaboration, communication, and documentation skills.
